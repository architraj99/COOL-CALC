# COOL CALC

COOL CALC is a calculator website that combines three different calculators in one place. 
Instead of opening separate tools, you can choose the calculator you need from the homepage and start using it right away.

 The website includes a Basic Calculator for everyday calculations, a Scientific Calculator for more advanced math , and a BMI Calculator for checking body mass index.

## Features

### Basic Calculator

* Addition, subtraction, multiplication, and division
* Percentage calculations
* Square root
* Keyboard support
* Calculation history
* Dark mode

### Scientific Calculator

* sin, cos, and tan
* log
* π and e constants
* Square root
* x²
* Calculation history
* Dark mode

### BMI Calculator

* Calculate BMI using height and weight
* Shows BMI category based on the result

## Built With

* HTML
* CSS
* Java

## How to Run

1. click on thee link
